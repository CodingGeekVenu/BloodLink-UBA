# BloodLink – Emergency Blood Donor Network

## Overview
BloodLink is an offline-first emergency blood donor coordination app for rural and semi-urban areas.

## Problem
Existing blood donor systems often depend on internet connectivity, static contact lists, or blood bank availability searches. Rural emergencies need verified, live, local donor coordination.

## Proposed Solution
BloodLink connects donors, acceptors, and health workers using:
- Cloud-based matching when internet is available
- Local offline storage
- BLE-assisted nearby donor discovery
- Health-worker verification
- Request status tracking

## User Roles
- Donor
- Acceptor / Patient Family
- Health Worker / Hospital Staff
- Admin

## MVP Features
- Donor registration
- Availability status
- Emergency request creation
- Donor matching
- Request tracking
- Health-worker verification
- Offline mode prototype
- BLE proof-of-concept placeholder
